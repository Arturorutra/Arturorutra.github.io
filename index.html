Welcome to your visit.I am leranning Markdown form Github.If there are some erros,please correct me.Thank you!

Background

Advantages

  • Focus on text content
  • Plain text, easy to read and writr, can be easily inconrporated into version control
  • The grammar is simple, there is no learning cost, and it is easy to make beautiful and generous typesetting at the same time of code words

Scences to be used

  • Various code hosting platforms

    Mainstream code hosting platforms suchs as GitHub, GitLab, BitBucket,Coding, Gitee, etc. All support Markdown syntax. MAny open sourcs project READMEs, development doucumenation,help documentation, wikis, etc. All written in Markdown

  • Technical community and writting platform

    StackOverflow、CSDN、掘金、简书、GitBook、有道云笔记

  • Forum

    V2EX、光谷社区

    The personal regret is that the platform may use different lanaguages to implement the Markdown paising engine, or use different versions of the same parsing engine, and may have different levels of customization and extention, which leads to the experience of using Markdown writing on different platforms. Not exactly the same. Fortuantely, for everyone to recognize some of the standard grammar, each family is supported.
