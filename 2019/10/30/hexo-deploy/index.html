保存md文件

copy md文件，粘贴到你当初建的博客站点文件夹下的source\ _posts目录下，一个md文件对应一篇博客文章。

hexo cmd

  1. 在博客站点文件夹下右键空白处，选择Git Bash Here。
  2. hexo generate 生成静态页面，
  3. hexo server 到localhost:4000预览博客效果
  4. hexo deploy 同步到github
